Backbone Dropdown

When adding an item in this app you should be able to pick a Priority.

Tasks

  1. Add API calls and rendering to js/views/addItemView.js so that the <select> contains the priority options.
  2. Use the App.api.getUserSettings to obtain the user settings and use the defaultPriority field to preselect that priority in the form. This requiries you to load both the priorities and the settings before rendering the form. Have a look at Q.all.
